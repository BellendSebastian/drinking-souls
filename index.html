Drinking Souls no flash version

Just a little info about this. This is a quick and dirty JavaScript prototype of preparetodrink.com.

Created by Bellend Sebastian in the space of about 2 hours or something.

How this works. It uses localStorage to store your previous search and repopulates the form based on that saved setting (or uses a clear form if no settings found). Filling out the form and submitting it will update the stored query string. So if you come back to this page in the same browser in future, you should get the same results as last time.
